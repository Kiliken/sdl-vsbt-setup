# sdl-vsbt-setup
A minimal setup for building SDL3 projects using only the Visual Studio Build Tools (VSBT) from the command line.

```powershell
curl.exe -L -o "$(Get-Location)\codeDL.ps1" "https://raw.githubusercontent.com/Kiliken/sdl-vsbt-setup/main/codeDL.ps1"
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "$(Get-Location)\codeDL.ps1"

```
